# yt_tk_api
dropshipping api connection
go to app.py, modify the tk_app_key, tk_app_secret, yt_user_account, yt_token with the credencail provided
install any required package via pip in terminal
run app.py using command line argument python app.py
go to link: https://services.tiktokshops.us/open/authorize?service_id=7347592207432337198
log in with your tiktok shop credential via tiktokshop seller center
it will redirect you to localhost/8081/callback
go back to terminal
enter the number of package just shipped.
it will automatically send outbound bill to yt warehouse.
